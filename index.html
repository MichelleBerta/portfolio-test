  • descriptionPassword Generator

    This application is a random password generator. Using prompts the generator asks the user five questions about the characters to be included.
    1. how many characters, between 8 - 128,
    2. use lower case letters,
    3. use upper case letters,
    4. use numbers, and
    5. use special characters.
    All of the desired characters are pulled from the ASCII character table. Through prompts and confirms, the users responses are stored. Using a for loop, js pulls the needed number and type of characters. The js file looks to the html file for when to begin generating the password and where to display it once it is created.

    Generate a Password
